What the script does:
*********************
This script is made for enabling alternative search engines to the Jolla browser.

A guide how to enable DuckDuckGo as search engine for the browser is available here:
http://talk.maemo.org/showthread.php?t=92129


Future development ideas:
*************************

There is a patch installation system called patchmanager:
https://openrepos.net/content/sfietkonstantin/patchmanager

It would be great to provide the custom search engines as a patch for this.
